# Online Courses Catalog

Тестовое задание. Каталог курсов с фильтрацией по категориям и живым поиском.

## Демо

[Ссылка на Vercel](https://kozyrev-test-phi.vercel.app/)

## Функционал

- Адаптивная резиновая вёрстка (1920px → 320px )
- Фильтрация курсов по категориям
- Живой поиск по названию
- Кнопка "Load more" (9 → 17 курсов)
- по макету Figma

## Технологии

- HTML5
- SCSS (препроцессор)
- JavaScript (Vanilla)
- BEM-методология


## Запуск локально

### Установка
1. Склонируй репозиторий
2. Открой проект в VSCode

### Просмотр
- Просто открой `index.html` в браузере

## Адаптив

- Плавное масштабирование от 320px до 1920px
- Использование `clamp()` и `calc()` для резины
- На мобильных устройствах декорации скрываются
- Грид-сетка сама подстраивается под ширину экрана

## Детали реализации

- **Категории:** Marketing (4), Management (3), HR & Recruiting (5), Design (2), Development (3)
- **Фильтрация:** при клике на категорию показываются только курсы этой категории
- **Поиск:** работает по названию курса в реальном времени
- **Load more:** сначала 9 курсов, потом ещё 8
